How it works
  • 1
    A random AES-256-CBC key is generated for every encryption.
  • 2
    Your PDF is encrypted with this AES key instantly.
  • 3
    The AES key is wrapped with the recipient's RSA-2048 public key (OAEP/SHA-256).
  • 4
    Both are bundled into a .enc JSON envelope and downloaded.
  • 5
    Only the recipient's private key can unlock the AES key and recover the PDF.
RSA-2048 OAEP + AES-256-CBC  ·  No files stored on server — all processing is in-memory
